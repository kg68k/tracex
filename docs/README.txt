* はじめに

  このプログラムは Human68k の DOS CALL を監視し、DOS CALL 名、引数、
  及び結果を標準出力( あるいは指定したファイル )に出力します。なにかの
  役には立つでしょう。


* 配布内容

	README.DOC	このドキュメント
	trace.1		マニュアル
	trace.x		バイナリ
	trace_src.Lzh	ソースファイル一式


* 実現法

  ソースを見て下さい。F line trap を乗っ取っています。trace の F line
  trap のハンドラは大部分が HUMAN 内部のハンドラと同じです。:-) 最初は 
  DOS CALL の実行はオリジナルのハンドラに任せる予定だったのですが・・・
  うまくいかなかったもので・・・


* 著作権

  慮願います。
